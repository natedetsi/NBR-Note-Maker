      Instructions: Enter the oldest date into the first data field, and then either todays date or another date you are using. Note use the Tab key to index between sections, you can also use the Enter key to activate all buttons.

      Interest calculator

      Instructions: This calculator is intend to be used to calculate the amount of interest you need to spread over 𝑥 amount of installments.

      First: input the value of the first interest rate into field one, then the second rate into the second field this will give you the difference, then tab to the installment field. You should now see a difference calculation.

      Second: enter the number of installments you need to spead over, and click calculate. Below you will see the amount per installment that you need to input.

      Third: Below this is a remainder amount, this is the amount you will need to add or subtract from one or more installments to reach the rounded amount of the difference. It is recommended to spread this over multiple installements.
